## 项目简介

**一个基于arduino和processing平台开发的交互类解压小游戏.**
**通过控制arduino板块的摇杆完成由processing界面中的任务.**

------

## 游戏规则

**通过操控摇杆来控制白色大球的运动，白色大球可以驱赶小球们前进。当小球跑进中央大球时，小球就会被大球吃掉，消失不见，直至所有小球都消失。**

------
## 项目亮点
+ 小球的尾巴由水墨效果构成，能够相互影响交融.
+ 中央大球吸收小球之后会变成冰蓝色.
+ 画面效果赏心悦目，能够缓解压力.
+ 能够活动长时间操控鼠标的手.
